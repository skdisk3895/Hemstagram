---------- 홈페이지 들어가는 과정 -------------------
1. 바탕화면에 있는 hemstaram 폴더에 들어간다.
2. 백색화면에(폴더에 하면 안됨) 마우스 우클릭을 클릭한 다음에 'Code로 열기'를 누른다.
3. git pull origin master 로 파일을 load한다.
4. 홈페이지를 열려면 visual code 프로그램의 하단부에 터미널에 'python manage.py runserver'를 친다. (안대면 문의하셈)
5. ctrl + 마우스클릭으로 URL(127.00.00 ...)클릭
6. 볼 일이 다 끝나면 터미널창에서 crtl + c 로 서버를 종료한다

------------- 저장하는과정(무엇인가 변화가 있을 때) -------------------
1. git add .
2. git commit -m '하고싶은 말'
3. git push origin master
